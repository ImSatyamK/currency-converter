# Currency Converter

A simple web app that converts currencies using live exchange rates.

**Live Demo:** https://statuesque-buttercream-6c5228.netlify.app

## Features
- Convert between multiple currencies
- Live exchange rates from a public API
- Dynamic country flags for selected currencies
- Instant conversion on input/change
- Default conversion: USD → INR

## Tech Stack
- HTML
- CSS
- JavaScript
- Netlify (deployment)

## APIs Used
**Currency API**
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api

Example:
https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api@latest/v1/currencies/usd.json
